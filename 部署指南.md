# 🚀 部署指南 - HONG KONG COOKIES TRADING LIMITED

## 部署架构

```
前端 (Vercel 免费)  ←→  后端 (Railway 免费)
  https://your-shop.vercel.app     https://your-api.railway.app
```

---

## 第一步：推送代码到 GitHub

### 1.1 注册 GitHub（已有账号跳过）
- 打开 https://github.com
- 点击 Sign Up 注册

### 1.2 创建仓库
1. 登录 GitHub 后点击右上角 "+" → "New repository"
2. 填写：
   - Repository name: `smoke-shop` （随便取）
   - 选择 **Public**（免费方案）
   - 不勾选 Initialize README
3. 点击 **Create repository**

### 1.3 本地推送代码（在命令行运行）

```bash
cd C:\Users\Administrator\WorkBuddy\20260411184422

git init
git add .
git commit -m "Initial commit - Smoke Shop Website"
git remote add origin https://github.com/YOUR_USERNAME/smoke-shop.git
git branch -M main
git push -u origin main
```

> 把 `YOUR_USERNAME` 替换成您的 GitHub 用户名

## 第二步：部署后端到 Railway

### 2.1 注册 Railway
1. 打开 https://railway.app
2. 点击 **Login with GitHub**（用刚才的 GitHub 账号登录）
3. 免费额度：每月 $5 美元（约 500小时），够用

### 2.2 创建后端服务
1. 登录后点击 **New Project**
2. 选择 **Deploy from GitHub repo**
3. 选择您的 `smoke-shop` 仓库
4. Railway 会自动检测，点击 **Deploy**

### 2.3 配置后端设置
部署完成后，在 Railway 项目页面：

1. 点击服务 → **Settings** → **Service** → Root Directory 设为：
   ```
   backend
   ```
2. 点击 **Variables** → 添加环境变量：
   ```
   NODE_ENV=production
   ```
3. 复制后端 URL（格式：`https://xxxx.railway.app`）

---

## 第三步：部署前端到 Vercel

### 3.1 注册 Vercel
1. 打开 https://vercel.com
2. 点击 **Sign Up with GitHub**

### 3.2 导入项目
1. 点击 **Add New Project**
2. 选择您的 `smoke-shop` GitHub 仓库
3. 配置：
   - **Framework Preset**: Vite
   - **Root Directory**: `frontend`
   - **Build Command**: `npm run build`
   - **Output Directory**: `dist`

### 3.3 设置环境变量
在部署设置中添加：
```
VITE_API_BASE_URL = https://你的railway后端地址.railway.app/api
```

4. 点击 **Deploy**

---

## 第四步：更新 CORS 配置

部署完成后，获取 Vercel 的前端地址（如 `https://smoke-shop.vercel.app`），
回到 Railway 后端，在 Variables 中添加：
```
FRONTEND_URL = https://smoke-shop.vercel.app
```

## ⚠️ 注意事项

1. **图片存储**：Railway 免费版的文件系统是临时的，重启后图片会丢失
   - 建议：将产品图片上传到 [ImgBB](https://imgbb.com)（免费图床），复制链接填入产品
   - 或者升级到 Railway Pro（$5/月）获得持久存储

2. **数据库持久化**：Railway 的磁盘数据在重新部署时会保留，但免费版偶尔可能重置
   - 建议：定期从管理后台导出数据备份

3. **免费额度**：
   - Railway：每月 $5 免费额度（够用）
   - Vercel：永久免费（静态网站）
