## 二、核心功能

### 2.1 客户端（前台购物网站）

1. **产品浏览**
   - 产品列表页（网格布局，响应式：桌面4列/平板2列/手机1列）
   - 产品详情页
   - 分类筛选

2. **变体选择器**（产品详情页）
   - 颜色（Color）- 必选
   - 尺寸（Size）- 必选
   - 规格（Specification）- 必选
   - 材质（Material）- 必选
   - 自定义参数1（选填，可自定义名称和值）
   - 自定义参数2（选填，可自定义名称和值）
   - 自定义参数3（选填，可自定义名称和值）
   - 备注（选填）

3. **购物车**（侧边滑出式）
   - 显示已选产品：图片、名称、变体信息、单价、数量
   - 可修改数量、删除产品
   - 显示总价

4. **客户信息表单**（购物车底部）
   - 姓名（Name）
   - 地址（Address）
   - 电话（Phone）- 选填
   - 邮编（Postal Code）

5. **PDF订单生成**（核心功能！）
   - 点击"确认订单"按钮生成PDF并下载到本地
   - PDF内容包含：
     - 公司名称：HONG KONG COOKIES TRADING LIMITED
     - 订单日期
     - 客户信息（姓名、地址、电话、邮编）
     - 产品明细表：名称、SKU、颜色、尺寸、规格、材质、自定义参数、数量、单价、小计
     - 订单总金额

6. **中英文切换**
   - 导航栏右上角语言切换按钮
   - 所有页面文字支持中英文

7. **不需要支付系统**
   - 只是生成订单PDF，不做在线支付

---

### 2.2 管理端（产品上传后台）

1. **产品管理**
   - 最多20个主产品
   - 每个产品最多500个SKU
   - 支持添加/编辑/删除产品

2. **产品信息字段**
   - 产品名称（中英文）
   - 品牌
   - 描述（中英文）
   - 价格（美元）
   - 分类
   - 产品图片（**本地上传**，非URL输入）
   - 支持格式：jpg, png, gif, webp
   - 最大10MB

3. **变体/SKU管理**
   - 颜色选项（多选或逗号分隔输入）
   - 尺寸选项
   - 规格选项
   - 材质选项
   - 3个自定义参数（名称+值，选填）
   - SKU前缀
   - **自动生成组合**：根据选择的颜色×尺寸×规格×材质自动计算SKU数量
   - 例如：3色×3尺寸×2规格×2材质 = 36个SKU自动生成

## 四、技术栈建议

### 前端
- React + TypeScript + Vite
- Tailwind CSS
- react-router-dom（路由）
- jsPDF（PDF生成）
- lucide-react（图标）

### 后端
- Node.js + Express
- SQLite（sql.js 或 better-sqlite3）
- multer（图片本地上传）
- cors

### 数据库表
```sql
-- 产品表
CREATE TABLE products (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  name TEXT NOT NULL,
  name_en TEXT,
  brand TEXT,
  description TEXT,
  description_en TEXT,
  price REAL NOT NULL,
  image_url TEXT,
  category TEXT,
  colors TEXT,          -- 可选颜色，逗号分隔
  sizes TEXT,           -- 可选尺寸，逗号分隔
  specifications TEXT,  -- 可选规格，逗号分隔
  materials TEXT,       -- 可选材质，逗号分隔
  custom1_name TEXT,
  custom1_values TEXT,
  custom2_name TEXT,
  custom2_values TEXT,
  custom3_name TEXT,
  custom3_values TEXT,
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 产品变体表
CREATE TABLE product_variants (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  product_id INTEGER NOT NULL,
  sku TEXT UNIQUE NOT NULL,
  color TEXT,
  size TEXT,
  specification TEXT,
  material TEXT,
  custom_param1_name TEXT,
  custom_param1_value TEXT,
  custom_param2_name TEXT,
  custom_param2_value TEXT,
  custom_param3_name TEXT,
  custom_param3_value TEXT,
  notes TEXT,
  price_modifier REAL DEFAULT 0,
  stock INTEGER DEFAULT 100,
  image_url TEXT,
  FOREIGN KEY (product_id) REFERENCES products(id)
);

-- 订单表
CREATE TABLE orders (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  order_number TEXT UNIQUE NOT NULL,
  customer_name TEXT NOT NULL,
  customer_address TEXT NOT NULL,
  customer_phone TEXT,
  customer_postal_code TEXT,
  total_amount REAL NOT NULL,
  status TEXT DEFAULT 'pending',
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 订单明细表
CREATE TABLE order_items (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  order_id INTEGER NOT NULL,
  product_name TEXT NOT NULL,
  variant_sku TEXT NOT NULL,
  color TEXT,
  size TEXT,
  specification TEXT,
  material TEXT,
  custom_params TEXT,
  quantity INTEGER NOT NULL,
  unit_price REAL NOT NULL,
  subtotal REAL NOT NULL,
  FOREIGN KEY (order_id) REFERENCES orders(id)
);
```

### API接口
```
GET    /api/products              获取产品列表
GET    /api/products/:id          获取产品详情（含变体）
POST   /api/products              创建产品
PUT    /api/products/:id          更新产品
DELETE /api/products/:id          删除产品

POST   /api/variants/batch        批量创建变体
POST   /api/variants/generate-combinations  生成SKU组合
GET    /api/products/:id/variants 获取产品变体
PUT    /api/variants/:id          更新变体
DELETE /api/variants/:id          删除变体

POST   /api/orders                创建订单
POST   /api/upload/image          上传产品图片（本地文件）
POST   /api/upload/images         批量上传图片
```
